# SIEM-System
A Security Information Event Management (SIEM) system was created in C++ for monitoring security events such as, tracking unsafe IPs, blocking malicious activities and searching users within a system to check their state.

## Features
- Alert Detection: The system monitors login events and triggers alerts
- Event Processing: Loads security events from external files
- User Search: Ability to check security status of users
- Interactive Menu: Simplistic command line menu for system management
- IP Management: The ability to block IPs
